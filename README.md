# WaifuPls
WaifuPls is a fun discord bot that sends generates random/daily waifus, anime, and more to come

# Commands
### Default prefix: ```*```
| Command  | Description |
| :---  | :---  |
| waifupls  | Sends a random waifu  |
| setprefix  | (MOD ONLY) Sets the bot's prefix  |
